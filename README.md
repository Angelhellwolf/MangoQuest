# MangoQuest
The unique & amazing Quest plugin for Spigot.

Maintained for Spigot/Paper 1.20.x and 1.21.x.

Build with:

```sh
mvn -DskipTests package
```

The shaded plugin jar is generated at `target/MangoQuest-1.10.0.jar`.
